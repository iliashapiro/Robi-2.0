tm1-ui-rpt-row-label
directive in module tm1Ui

Description

Use this to format and enable expanding/collapsing of your elements per row dimension. This directive is used in the tbody section of your table element with the active form. Use as part of tm1-ui-rpt-view.

Usage

as element:
<tm1-ui-rpt-row-label
       tm1-row-data="{object}"
       tm1-dimension="{string}"
       [tm1-freeze-element="{boolean}"]
       [tm1-ui-class="{string}"]>
</tm1-ui-rpt-row-label>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
tm1-row-dataobject

The row property of the object defined on the parent tm1-ui-rpt-view. REQUIRED

tm1-dimensionstring

The dimension that will be referred to when expanding/collapsing. REQUIRED

tm1-freeze-element
(optional)
boolean

To disable collapsing/expanding on this element row

(default: "false")

tm1-ui-class
(optional)
string

Add other custom classes into this directive.

(default: "tm1-ui-rpt-row-label")
